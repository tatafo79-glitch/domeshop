상품 이미지 처리 예시

.env DB 연결로 읽은 현재 등록설정 기준입니다. contain 썸네일은 설정 크기 흰색 캔버스 안에 원본 전체가 들어가도록 생성합니다.

원본 검증 600 x 600px 정사각 기준 / 20MB
목록이미지 300px / contain
상세이미지 500px / contain
상세목록 / 기타목록 50px / 80px
플랫폼용 이미지 1000 x 1000px / contain 생성
